# SRadioboxComponent

Extends **SInputWebComponent**

Provide a nice and easy to customize radio and checkbox webcomponent extension

### Example
```html
	<label>
	<input type="checkbox" is="radiobox" /> Hello
</label>
<label>
	<input type="radio" is="radiobox" /> World
</label>
```
